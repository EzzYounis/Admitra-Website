# Admitra Information Page

Modern one-page website built with React + Vite to present:

- what Admitra does,
- why Admitra exists,
- app screenshots,
- App Store and Google Play download actions.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Replace branding assets

- Replace logo file: `public/branding/admitra-logo.svg`
- Replace screenshots:
	- `public/screenshots/app-screen-1.svg`
	- `public/screenshots/app-screen-2.svg`
	- `public/screenshots/app-screen-3.svg`

You can also tune the theme colors in `src/index.css` (`--brand-primary`, `--brand-secondary`, `--brand-deep`) to match the official logo palette.
